# Bitácora de cambios del proyecto

Este documento se actualiza con cada modificación relevante al proyecto para entregar el informe mensual solicitado.

## 2025-12-18
- Se crea esta bitácora para registrar los cambios de desarrollo y validación.
- Se reforzó el flujo de carga masiva: el área de arrastre y el botón de archivo se habilitan solo con un correo válido, se muestran mensajes de validación (incluida la fecha disponible +4 días) y el checklist inicial bloquea envíos cuando detecta inconsistencias.
- El guardado local sigue disponible únicamente cuando la validación concluye sin errores y con credenciales consistentes.
